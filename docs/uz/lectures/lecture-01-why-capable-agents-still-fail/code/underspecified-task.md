# Yetarlicha Ta ľriflanmagan Vazifa Misoli

AI savol-javob imkoniyatiga ega desktop bilimlar bazasi (knowledge base) ilovasini quring.

Cheklovlar:

- Hech narsa ko Ľrsatilmagan
- Ishga tushirish (startup) buyrug Ľi berilmagan
- Jild strukturasi bo Ľyicha ko Ľrsatma yo Ľq
- Ma ľlumotlar modeli aniqlanmagan
- Aniq tugatish (completion) mezonlari yo Ľq

Bunday promptdan odatiy kutiladigan natijalar:

- agent o Ľzidan o Ľzi ixtiyoriy tuzilmani o Ľylab topadi
- ilova komplyatsiya bo Ľlishi mumkin, lekin barqaror ishga tushmasligi mumkin
- UI foydalanish mumkin bo Ľlgan kiritish/so Ľrov (ingest/query) yo Ľllari hosil bo Ľlmasidan paydo bo Ľlishi mumkin
- agent ko Ľpincha kosmetik muvaffaqiyatdan keyin to Ľxtaydi
