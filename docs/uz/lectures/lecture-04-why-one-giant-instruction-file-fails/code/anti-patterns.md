# Yo Ľriqnoma Fayli Antipattern ľlari (Xato Usullari)

- Barcha repozitoriy bilimlarini bitta faylga joylashtirish
- Ayni bitta qoidani turli joylarda takrorlash
- Hech kim tekshirmaydigan eskirgan qoidalarni kodlashtirish
- Juda kam qo Ľllaniladigan shartli yo Ľriqnomalarni yozish
- Katta vosita qo Ľllanmalarini (tool manuals) ishga tushirish kontekstiga tiqib yuborish
